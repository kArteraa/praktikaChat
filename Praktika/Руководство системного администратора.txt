Функционал сайта:
	
	DeepPavlov ( модель Named Entity Recognition (NER)), ChatGPT


Как запустить сайт:

	1) Скачать проект. Ссылка на репозиторий - https://github.com/kArteraa/praktikaChat
	2) Скачать локальный сервер. Мы использовали OpenServer. Ссылка на скачивание с официального сайта - https://ospanel.io/download/
		2.1) В настройках модулей поставить PHP 8.1, Apache_2.4-PHP_8.0-8.1
	3) Скачать Python, при скачивании нужно нажать галочку Add to PATH (Версии Python 3.6, 3.7, 3.8, 3.9, 3.10 , версии ниже или выше не поддерживаются)
	4) Скачать библиотку DeepPavlov и модель Named Entity Recognition (NER).
		4.1) Зайти в командную строку Windows
		4.2) Установить - pip install deeppavlov
		4.3) Скачать модуль - python -m deeppavlov install ner_ontonotes_bert_mult
		4.4) Установить модуль - python -m deeppavlov download ner_ontonotes_bert_mult
		4.5) Запустить REST сервер - python -m deeppavlov riseapi ner_ontonotes_bert_mult -p 5005
	
	5) Добавить папку с проектом в папку domains OpenServer'a
	6) Запустить сайт
	


Работа с сайтом, описание функционала сайта и описание возможных ошибок:
	
	На главной странице сайта находятся ссылку на ChatGPT и DeepPavlov.
	
	При переходе на страницу с ChatGPT находится интерфейс с чатом. Можно написать любой вопрос и получить ответ на него. Возможные ошибки: если написать не вопрос, то ChatGPT может ответить рандомное выражение или текст, так-что нужно задавать именно вопросы.
	
	При переходе на страницу с DeepPavlov находится интерфейс с чатом. Мы использовали модель Named Entity Recognition (NER), что классифицирует слова текста по типу именованных сущностей, к которым они принадлежат (имена собственные, географические названия). Возможные ошибки: иногда результат работы модели не обрабатывается, и на вход мы получаем наш текущий текст.
